Dynamic Directives

Dynamic Directives are a set of directives for angularJS that enable developers to quickly create and use dynamic elements. Examples I can think of are: embedded chatrooms and comment sections but there are probably many more uses.

Dynamic Directives rely's on CORS (Cross Origin Resource Sharing) which allows you to create cross-origin AJAX requests.

Contact Form Example

This example uses the dd-contact directive to send an email to the specified address. This example uses a cross-origin AJAX request to my personal server located at cors.emaf.ca.

In this example, I am using the dd-contact directive embedded within the dd app div which creates the contact form shown under Output. By filling in your email address into the field shown below, angular updates the DOM to change the ddid attribute of the dd-contact directive to update the address that the email will be sent to.

To test this example, enter your email in the field with placeholder "your email", you should see the ddid attribute of dd-contact to update respectivly. Next, fill out the contact form and click send. You should see an email in your inbox formatted like the preview.

Code:

    <div ng-app="dd">
    <dd-contact ddid="{{adminEmail}}"></dd-contact>
    </div>
